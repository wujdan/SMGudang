# 📦 Sistem Manajemen Barang (Inventory System)

Aplikasi berbasis Laravel untuk mengelola data barang, transaksi barang masuk dan keluar, serta pembuatan laporan secara terstruktur.

---

## 🚀 Fitur Utama

### 🔐 Autentikasi

- Login & Logout user
- Proteksi halaman menggunakan middleware

---

### 📦 Manajemen Barang

- CRUD data barang
- Pencarian barang (search)

---

### 📥 Barang Masuk

- Input data barang masuk
- Menambah stok secara otomatis

---

### 🛠️ Transaksi Pekerjaan

- Membuat data pekerjaan
- Menambahkan item ke dalam pekerjaan (seperti cart)
- Mengelola barang keluar
- Fitur return barang

---

### 📊 Laporan

- Laporan stok barang
- Laporan barang masuk
- Laporan barang keluar
- Statistik penggunaan barang
- Rekap data transaksi

---

## 🏗️ Arsitektur

Project ini menggunakan pola MVC (Model-View-Controller) dari Laravel:

- **Model** → Representasi data & database
- **View** → Tampilan (Blade)
- **Controller** → Logic aplikasi

---

## 🛠️ Teknologi

- PHP (Laravel Framework)
- MySQL / MariaDB
- Blade Template Engine
- Bootstrap (opsional UI)

---

## 📂 Struktur Modul

- `Barang` → Master data barang
- `BarangMasuk` → Data stok masuk
- `Pekerjaan` → Transaksi barang keluar
- `Laporan` → Reporting & statistik
